# Project Rules

- **Strictly Free Services Only**: Under no circumstances should you recommend, configure, or depend on paid services (e.g., AWS, Hetzner, OpenAI, Anthropic Claude). Always use 100% free tiers (like Render, Vercel, Neon DB, Upstash Redis) or local open-source solutions. If a task requires a paid service, propose a free alternative or ask the user if they want to skip it.
